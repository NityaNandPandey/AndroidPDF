public MenuItem setIconTintList (ColorStateList tint)

Applies a tint to this item's icon. Does not modify the current tint mode, which is SRC_IN by default.

Subsequent calls to setIcon(Drawable) or setIcon(int) will automatically mutate the icon and apply the specified tint and tint mode using setTintList(ColorStateList).

Parameters
tint the tint to apply, may be null to clear tint
